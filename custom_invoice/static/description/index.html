Adapta el terminal punto de venta para realizar facturas electrónicas en México (CFDI 3.3)

Se pueden emitir facturas:

  • Por venta
  • De ventas de un periodo (día, semanas, mes)
  • 1) Con un solo concepto
  • 2) De todas los productos


  • Una factura engloba múltiples órdenes de venta.


  • Factura con múltiples órdenes de venta y un solo concepto


  • Factura con múltiples órdenes de venta y todos los conceptos.


  • Información para timbrar la factura.


  • Datos del receptor.


  • Datos del emisor.


  • Factura timbrada.


  • Se anexa el archivo xml y pdf para enviar correo electrónico.
